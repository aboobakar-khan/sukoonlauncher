# Pomodoro Completion Sound

## Sound File

Place a soothing completion sound file named `pomodoro_complete.mp3` in this directory.

### Recommended Sound Characteristics:
- **Duration**: 2-5 seconds
- **Type**: Gentle bell, chime, or singing bowl sound
- **Volume**: Soft and calming (not jarring)
- **Format**: MP3 or OGG
- **Purpose**: Signal the end of a focus/break session without being disruptive

### Free Sound Resources:
- **Freesound.org**: Search for "meditation bell", "zen chime", "singing bowl"
- **Zapsplat.com**: Free sound effects
- **Mixkit.co**: Free sound effects

### Example Searches:
- "Tibetan singing bowl single note"
- "Meditation bell gentle"
- "Zen chime soft"
- "Crystal bowl ring"

## Alternative Implementation

If you don't add a sound file, the app will fallback to:
- Double haptic feedback (vibration)
- Visual celebration animation on screen

The Pomodoro timer will work perfectly without the sound file!
